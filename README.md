# SkillSphere

A modern online learning platform where students can discover, enroll in, and complete courses while instructors can share their expertise with the world.

## Purpose

SkillSphere is designed to make quality education accessible to everyone. It provides a seamless learning experience with course management, user authentication, progress tracking, and an intuitive interface that works beautifully on all devices.

## Key Features

**For Learners**: Browse through a curated collection of courses across different categories and skill levels. Enroll in courses that match your learning goals and access course content, curriculum modules, and detailed course information. Track your learning progress and manage your profile to personalize your learning experience.

**For Instructors**: Share your knowledge by publishing courses with structured curriculum modules. Reach a global audience of eager learners and build your reputation as an expert instructor.

**For Everyone**: Dark mode and light mode themes for comfortable learning at any time of day. Secure authentication system to keep your account and learning data safe. Responsive design that adapts seamlessly from mobile phones to desktop computers. Real-time session management that keeps you logged in and updated across all pages.

## NPM Packages Used

**Next.js** powers our full-stack application with server-side rendering and API routes for optimal performance.

**React** provides the interactive UI components that make the learning experience smooth and responsive.

**Tailwind CSS** enables us to style the application with a utility-first approach for consistent design and rapid development.

**HeroUI** gives us a beautiful set of pre-built components that maintain consistency and reduce development time.

**Better Auth** handles all authentication logic securely, including email signup and login functionality.

**Better Auth Mongo Adapter** integrates our MongoDB database seamlessly with the authentication system.

**MongoDB** stores all our application data including users, courses, and enrollment information.

**Framer Motion** creates smooth animations and transitions that delight users and enhance the visual experience.

**Lucide React** provides a comprehensive set of beautiful, consistent icons throughout the application.

## Getting Started

To run the development server on your machine, execute the following command:

```bash
npm run dev
```

After running this command, open [http://localhost:3000](http://localhost:3000) in your browser to see the application in action. The application will automatically reload as you make changes to the code.

## Building for Production

When you're ready to deploy, build the application with:

```bash
npm run build
npm start
```
